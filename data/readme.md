## 설명

- `level1`: 가장 약한 왜곡 수준의 캡차 이미지
- `level2`: 중간 수준의 왜곡이 적용된 캡차 이미지
- `level3`: 강한 왜곡이 적용된 캡차 이미지
- `level4`: 극한 수준의 왜곡이 적용된 캡차 이미지

## 파일 규칙

- 파일 형식: `.png`, `.txt`
- 파일명은 가능하면 문자열 라벨을 포함하도록 작성합니다.
- txt파일 안에는 문자열 정답만 작성하여 저장합니다.
  - 예: `A3K7B.png`, `9X2Q1.jpg`

## 주의사항

이 폴더에는 샘플 이미지가 포함되며, 전체 학습 데이터는 별도로 관리할 수 있습니다.
